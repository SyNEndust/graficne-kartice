Zgradba

Sestavni deli grafične kartice

  • Ohlajevalne enote - najbolj pogosti ventilatorji in bakrene cevi.
  • Povezovalne enote na primer napajalni sistem in reža za SLI ali CrossFire povezavo).
  • Zunanje povezovalne enote - HDMI, DisplayPort, VGA.
  • GPU - ogromno manjših cpu, ki opravljajo le eno delo. Lahko tudi manipulira spomin.
  • Video spomin - za hitro shranjevanje in uporabo podatkov (večinoma VRam).
  • Video BIOS(basic input/output system) - vsebuje majhno količino programa za prvi zagon "setup" in izvaja kontrolo nad grafično kartico.
  • RAMDAC - včasih najpomembnejša komponenta zdaj pa skoraj neobstoječa, saj pretvarja digitalne signale v analogne za prikaz na katodnih ceveh, ki so jih uporabljaji stari monitorji, televizije.
  • Motherboard interface - povezovalni sistem med matično ploščo in grafično kartico. Trenutno najbolj uporabljen PCI Express (Peripheral Component Interconnect Express).

Terminologija

Ko iščemo grafične kartice se mnogokrat srečamo z veliko računalniškega žargona. Nekaj mogosto uporabljenih besed je:

  • Memory bandwith - nam pove koliko spomina in kako hitro lahko izkoristi RAM
  • Memory clock/speed - hitrost s katero se spomin prepisuje
  • Memory Interface width - je velikost podatkov, ki jih lahko prenesemo v enem “urnem ciklu” (večinoma 128, 256 ali 384 bit).
  • Cores - število procesnih enot (več kot jih, hitreje deluje saj si porazdelijo delo)
  • Base and boost clock - opisuje frekvenco s katero deluje grafična kartica. Boost clock je najvišja možna frekvenca pri kateri kartica deluje vendar le za kratek čas, saj je pri tem zelo obremenjena.
  • Display - resolucija in hitrost(FPS) s katero prikazuje slike (ne more biti višja od monitorja)
  • Vsi podporni sistemi kot so Ray tracing - zelo napredni način generiranja svetlobe, z namenom lifelike prikaza.
  • Anti Aliasing - način spreminjanja grafike z glajenjem ostrih robov.
  • FSAA - fullscene antialiasing (vzame višjekvalitetno sliko in jo poenostavi da se znebi ostrih robov)
  • VR podpora - poznamo non immersive (igrice v katerem kontroliramo drugo osebo, kot so Fifa), semi immersive (simulatorji letenja za treniranje pilotov), fully immersive - kar mislimo z besedo VR)
